# Quran — Verse of the Day

A Chrome extension that replaces your new tab page with a beautiful daily Quran verse — complete with Arabic text, translations in 15+ languages, tafseer, prayer times, and more.

> Currently under review on the Chrome Web Store. In the meantime, you can install it manually using the instructions below.

---

## ✨ Features

- 📖 **Daily verse** — a new Quran verse every day, or random on every tab
- 🌍 **15+ language translations** — English, Urdu, French, Bengali, Russian, and more
- 📚 **Tafseer** — commentary from Ibn Kathir and other scholars
- 🕌 **Prayer times** — auto location detection or manual city search with autocomplete
- 📅 **Gregorian & Hijri date** display
- 🎨 **Custom colour themes** — pick any colour to theme the whole page
- 🌙 **Dark mode** — auto-follows system preference with manual override
- 🖼 **Custom background image** — upload any photo
- ⭐ **Top sites shortcuts** — your most visited sites at a glance

---

## 🛠 Manual Installation (Developer Mode)

Until the extension is live on the Chrome Web Store, here's how to install it:

1. Download the latest zip from the [Releases](../../releases) page (or clone this repo)
2. Unzip the folder
3. Open Chrome and go to `chrome://extensions`
4. Enable **Developer mode** (toggle in the top right)
5. Click **Load unpacked**
6. Select the `verseoftheday` folder
7. Open a new tab ✨

> **Note:** A small "Quran — Verse of the Day" bar will appear at the bottom of the page. This is added by Chrome for all custom new tab extensions and cannot be removed.

## ⚠️ Known Issues
 
**"Quran — Verse of the Day" bar at the bottom of the page**
This is a Chrome limitation — whenever an extension overrides the new tab page, Chrome automatically adds a small label bar at the bottom. There is no way to remove it. It is a Chrome UI element, not part of the extension.

## 🔒 Privacy

No personal data is collected. All preferences are stored locally in your browser. Location is only accessed if you use the prayer times feature and is never sent to any server other than the prayer times API. See [PRIVACY.md](./PRIVACY.md) for full details.
